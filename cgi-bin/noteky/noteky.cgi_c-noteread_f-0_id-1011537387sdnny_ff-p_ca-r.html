夜中にふと目を覚まして、パソコンをつけてみたら掲載されていたので、読んじゃいました。明日も早いのに……。

さて、赤哉君。吸血鬼とハーフエンジェルとのハーフとのことで、吸血鬼の血が２分の１。天使と人間の血が、それぞれ４分の１というわけですね。なるほどなるほど。そして吸血鬼の血がわずかに天使の血よりも勝ってしまったが故に吸血鬼化してしまったと。

これでなぜＴＳしたのかはまだ分かりませんが、これで考えたことが１つ。もしかしてお姉さんの天子さんも、ＴＳしてたりして……。つまり本来、天子さんは男の子として生まれたのだけど、天子の血が優勢になってしまったが故に、天使化してついでにＴＳして女の子になったと……。赤哉君（赤奈ちゃん）が中２で、天子さんが２０才の大学生ならば、それだけで年齢差は６才〜７才。どうせ３才〜４才ぐらいの時の記憶なんてあやふやだから、天子さんが９才〜１０才のころに天使の血が優勢になってＴＳしたとすると、赤哉君が覚えていないのも不思議じゃないし、天子さんが、女の子になった赤哉君にあれほど萌えるのも頷けるかな……、と思ったのですが、どうやらそういう裏設定はなさそうですね。

この家族がバックにいる限り、赤奈ちゃんには安息の日々は来ないというのは自明に思えますが、何とかがんばって日常してほしいものですね。それでは続きを楽しみにしています。

追伸
　同じ吸血鬼として、そして妻と上の娘に対抗するため、父親が赤奈ちゃんに吸血鬼のなんたるかを教えつつ、溺愛しながら、愚痴をこぼしていく未来が見えるようです。（笑）
　彼にも是非がんばって欲しいですね。趣味は悪いけど、この四人のメンバーの中では比較的まともな人物らしいし……。

二伸
　今回は、私の作品にインスパイアされてとのこと。大変に名誉なことです。有難うございました。

既に旧運営委員からコメントが出ていますが・・・

　おそらくノインさんは何らかのオーサリングソフトで作成されたと思いますが、
それが相当に質の悪いものであった可能性が高いです。

　こちらで使ったのはMozillaです。運営委員さんもノインさんも自分の環境では
気が付かなかったと書かれていますが、これは使用しているブラウザがInternet
Exprorerであるためです。IEは文法規則の間違いに非常に寛容で相当ひどい文法
ミスがあっても「それっぽく」表示してしまいます。これは閲覧する側から見ると
大きなメリットですが、チェックをおこなうという目的では全く役に立ちません。

　Mozilla(またはNetscape6.x)は書かれた記述の通りに表示してしまうので、
ある意味非常に残酷な表示になってしまうのですが、これで正常に表示されるなら、
文法に関しての問題は殆どない(=どんなブラウザでも正しく表示される)といえます。

　ちなみに今回の問題はfontタグの閉じ忘れが原因です。＜font＞で始まったら
必ず＜/font＞で閉じる必要がありますが、代わりに＜font＞と書いてしまっている
ために大きい文字のままになっているわけです。その数およそ数十個。それさえ
修正すれば、正常に表示されます。

参考：
http://www.mozilla.gr.jp/webtest/
