ｍｋ８４２６ 様＞
　うんうん。でも実は微妙な点が先行公開版とはことなっているのですにょ。
　そういう所を探すのも楽しいかも。
　ちなみに『日東小女生徒の王子様『戸増頼香』』に関して、
　頼香は女の子に人気があるようでふ。ファン倶楽部も存在します。
　それ以前に香奈はああいった活発な女の子を可愛いと思っている模様。

ゆあさのみき 様＞
　女の子ってのは王子様を待っている物って事で（爆）
　とりあえず頼香は裏では『王子様』と呼ばれて、写真とかが取り引きされているようです。

ジャージレッド 様＞
　ホワイトイグレット2000は、名前こそ「ナイト2000」系ですが、
日本人が作ったこともあり、機能的にはマッハ号的です。
　尚、純白のボディーカラーに関しては、「アスラーダ」の線も捨てられないです。
　やっぱ「アスラーダ」って「マッハ号」+「ナイト2000」って気がしません？

　「超科学の出所が惑星連合らしい」
　コレについては、半分当たりです。
　部分的には「からめる＆もけの許可の元、果穂＞紳二経由」で技術が伝えられてます。
　ただしそれは全体のごく一部。
　それ以外に関しては、白鷺の独自開発＆連邦提供技術を独自発展させた物です。
　そんな関係もあって、「オーラブレード＆フェイザー」が「オーラロッド」になるなど、
　微妙な点で今後は「らいか大作戦」とは相違点が出てくると思われます。
