　ほふ…これはスーパーリリカルですね。いやほんと…リリカルものがこのごろ少なかったから、ていうかリリカルと現実ものって絶対数が少ないのかな…というわけでこれはいままでにない感じがします。ある程度お約束というかベタな展開をするのも、またよしって感じで、安心して読める良質な作品という印象を持ちました。ここまで読者にバレバレな感じというのもいいですよ。

　ところで、リリカルな作品はある程度絶望が大事かも…というわけで次回か次々回には葛藤があるのを希望しときます。
　それでは。
